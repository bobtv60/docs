### What is Upview?

Upview is the ultimate feedback collection platform designed specifically for Roblox game developers. Collect player feedback, track issues, and gain insights to improve your games with our powerful dashboard and easy-to-use API.

## Key Features

### In-Game Integration

- Lightweight Roblox module for easy integration
- Feedback UIs that match your game's style
- Non-intrusive feedback collection
- Advanced customization on how you ask for and track player's feedback

### AI-Powered Insights

- Automatic feedback categorization (Bug, Suggestion, Spam, Rude, Other)
- Intelligent priority scoring for bug reports
- Sentiment analysis for player feedback
- Powered by a custom-trained and self-hosted model

### Developer Tools

- RESTful API for programmatic access
- Webhook support for further integrations
- Rate limiting and security features

### Game Updates Tracking

- Automatic detection of game updates via the Upview module
- Correlation analysis between updates and feedback
- Version-specific feedback filtering
