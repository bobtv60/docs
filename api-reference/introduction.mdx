## Welcome to the UpView API

The UpView API provides programmatic access to your feedback data, allowing you to integrate UpView with your existing tools and workflows.

## Authentication

All API requests require authentication using your API key. Include your API key in the request headers:

```bash
curl -H "x-api-key: upv_xxxx_xxxx_xxxx_xxxx" \
     https://api.upview.dev/feedback
```

</Note>
